# PACIA · Guía UX

## Diseño centrado en las personas

Cada pantalla debe responder:

- ¿Qué debo hacer?
- ¿Qué cambió?
- ¿Cuál es la siguiente acción?

## Principios

- Mostrar solo información accionable.
- Reducir clics.
- Reducir carga cognitiva.
- Explicar cada recomendación.
- Priorizar estudiantes antes que documentos.

## Dashboard

Debe mostrar:
- tareas prioritarias
- documentos nuevos
- estudiantes críticos
- tiempo estimado
- próximos pasos
