# PACIA · Design System

## Componentes
- Botón primario azul.
- Botón secundario blanco con borde.
- Tarjetas con esquinas redondeadas.
- Formularios simples.
- Timeline para expedientes.

- Consistencia.
- Accesibilidad.
- Responsive.
- Bajo esfuerzo cognitivo.
