# NEXO PACI
# Manual de Línea Gráfica
## Versión 1.0

> Documento maestro de identidad visual para la plataforma NEXO PACI.

## Filosofía visual

NEXO PACI debe transmitir:

- Confianza
- Profesionalismo
- Cercanía humana
- Trabajo colaborativo

Debe sentirse como una plataforma educativa moderna, no como un formulario burocrático.

## Concepto visual

La identidad se inspira en los colores de los delantales y uniformes utilizados por:

- Educadoras diferenciales
- Educadoras de párvulos
- Docentes
- Psicólogos
- Equipos PIE
- Asistentes de la educación

Cada perfil utiliza un color de acento sin perder una identidad común.
