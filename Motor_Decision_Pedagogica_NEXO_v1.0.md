# Motor de Decisión Pedagógica NEXO

Documento técnico completo disponible también en formato Word.

**Principio rector:** Evidencia → Barrera → Currículum → Respuesta educativa → Indicador → Seguimiento.

El motor organiza información, construye perfiles funcionales, vincula evidencias con el currículum, propone respuestas graduadas, exige validación humana y realimenta las decisiones mediante seguimiento.
